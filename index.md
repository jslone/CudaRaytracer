# Overview
A real-time GPU raytracer created using CUDA and adaptive thread-block allocation and assignment of rays.

# Getting Started
```
git clone git@github.com:jslone/CudaRaytracer.git
cd CudaRayTracer
mkdir build && cd build
cmake ../src
make
```

# Running
```
./build/bin/application
```
